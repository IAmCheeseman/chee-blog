NPT

NPT is a dynamically typed programming language written in C#, styled to look like lua. You can find the source code on github.

Here's a couple snippets showing the syntax 

                
let std = import("std")
let str = import("string")

let i = 0
i += 1 // Just needed to flex on lua a bit

let j = [1, 2, 3]
std.println(j[0]) // Ah, I accidentally flexxed more :/

fun my_function(a, b)
    return a + b
end

std.println(str.format("my_function() resulted in '{}'", [str(my_function(1 + 2))]))

Hello world: 

                
let std = import("std")
std.print("Hello, world!")

It's not fast. Not at all. In fact, it's slower than Python. Some day I might rewrite NPT and make it compiled.

If you want to see more of the syntax, you can check out the examples on Github.
